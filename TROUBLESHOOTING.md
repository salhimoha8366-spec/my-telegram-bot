# 🎬 دليل استكشاف الأخطاء

## الأخطاء الشائعة والحلول

### 1. خطأ: BOT_TOKEN غير موجود

**الأعراض:**
```
ValueError: ❌ BOT_TOKEN غير موجود
```

**الحلول:**

#### الطريقة 1: متغير البيئة
```bash
export BOT_TOKEN="your_token_here"
python bot.py
```

#### الطريقة 2: ملف .env
```bash
cp .env.example .env
# عدّل .env
nano .env
# أضف: BOT_TOKEN=your_token_here
python bot.py
```

#### الطريقة 3: على الخادم
```bash
# Railway
railway config:set BOT_TOKEN="token"

# Heroku
heroku config:set BOT_TOKEN="token"

# Docker
docker run -e BOT_TOKEN="token" telegram-bot
```

### 3. خطأ: FFmpeg لم يتم العثور عليه

**الأعراض:**
```
FileNotFoundError: ffmpeg not found
```

**الحلول:**

#### Ubuntu/Debian:
```bash
sudo apt-get update
sudo apt-get install ffmpeg
```

#### macOS:
```bash
brew install ffmpeg
```

#### Windows:
```bash
# استخدم Chocolatey
choco install ffmpeg

# أو حمّل يدوياً من:
# https://ffmpeg.org/download.html
```

#### التحقق من التثبيت:
```bash
ffmpeg -version
```

### 4. خطأ: خطأ في تحميل TikTok

- الفيديو محمي
- حظر جغرافي

1. **تحقق من الحساب:**
   - اذهب لملف TikTok
   - تأكد من أن الحساب عام
   - غيّر الإعدادات إذا لزم

2. **جرب VPN:**
   ```bash
   # استخدم VPN مختلف
   # حاول من موقع جغرافي مختلف
   ```

3. **جرب رابط مختلف:**
   - نسخ الرابط من العنوان مباشرة
   - جرب فيديو من حساب مختلف

4. **انتظر وحاول لاحقاً:**
   - قد تكون هناك مشاكل مؤقتة
   - جرب بعد ساعات

### 5. خطأ: حجم الملف كبير جداً

**الأعراض:**
```
❌ حجم الملف (120 MB) يتجاوز الحد الأقصى (50 MB)
```

**الحلول:**

1. **اختر جودة أقل:**
   - استخدم 480p بدلاً من أفضل جودة
   - استخدم "صوت فقط" للموسيقى

2. **استخدم قص الفيديو:**
   - قص الأجزاء المهمة فقط
   - حمّل أجزاء منفصلة

3. **حمّل من منصة مختلفة:**
   - قد تكون جودة مختلفة

---

### 6. خطأ: استهلاك عالي للموارد

**الأعراض:**
- استهلاك CPU عالي جداً
- استهلاك RAM مرتفع
- البوت يتجمد

**الحلول:**

1. **تنظيف الملفات:**
   ```bash
   # حذف الملفات المؤقتة
   rm -rf downloads/*
   ```

2. **تقليل الحمل:**
   - قلل عدد التحميلات المتزامنة
   - أعد تشغيل البوت دورياً

3. **استخدم خادم أقوى:**
   - زد موارد الخادم
   - استخدم خطة أعلى على الاستضافة

---

### 7. خطأ: مشاكل الاتصال بـ yt-dlp

**الأعراض:**
```
ConnectionError: Connection refused
```

**الحلول:**

1. **تحقق من الإنترنت:**
   ```bash
   ping youtube.com
   ping tiktok.com
   ```

2. **حدّث yt-dlp:**
   ```bash
   pip install --upgrade yt-dlp
   ```

3. **استخدم proxy:**
   ```python
   # في bot.py أضف:
   "proxy": "http://proxy-server:port"
   ```

### 9. خطأ: مشاكل النشر على الخادم

**الأعراض:**
- الخادم يرفع الملف ثم يوقفه
- Deploy fails

**الحلول:**

#### Railway:
```bash

# أعد النشر
git push
```

#### Heroku:
```bash
# تحقق من السجلات
heroku logs --tail

# أعد النشر
git push heroku main

# أو
heroku ps:restart worker
```
