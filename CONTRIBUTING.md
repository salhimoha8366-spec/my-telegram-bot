## 🤝 كيفية المساهمة

شكراً لاهتمامك بالمساهمة في المشروع! 🎉

### خطوات المساهمة

#### 1. Fork المستودع
```bash
# انسخ المستودع إلى حسابك
# اضغط على زر Fork على GitHub
```

#### 2. استنساخ المستودع المحلي
```bash
git clone https://github.com/your-username/my-telegram-bot.git
cd my-telegram-bot
```

#### 3. إنشاء branch جديد
```bash
git checkout -b feature/my-feature
# أو
git checkout -b fix/my-fix
```

#### 4. تطوير وتعديل الكود
```bash
# أضف التحسينات أو الميزات
# اختبر التعديلات محلياً
python bot.py

# أو شغّل الاختبارات
python -m unittest test_bot.py
```

#### 5. Commit التعديلات
```bash
git add .
git commit -m "إضافة ميزة جديدة: وصف قصير"
```

#### 6. Push التعديلات
```bash
git push origin feature/my-feature
```

#### 7. فتح Pull Request
- اذهب إلى المستودع الأصلي
- اضغط على "New Pull Request"
- اختر branch الجديد
- اكتب وصف التعديلات

## 🐛 الإبلاغ عن الأخطاء

### خطوات الإبلاغ

1. **تحقق من أنه لم يتم الإبلاغ عنه:**
   - ابحث في Issues الموجودة

2. **افتح Issue جديد:**
   - اذهب إلى "Issues"
   - اضغط "New Issue"
   - اختر "Bug report"

3. **أكمل النموذج:**
   - **العنوان**: وصف مختصر للمشكلة
   - **الوصف**: شرح مفصل
   - **الخطوات**: كيفية إعادة الخطأ
   - **السلوك المتوقع**: ما يجب أن يحدث
   - **المعلومات**: النظام والإصدار

### مثال إبلاغ جيد:

```
العنوان: خطأ في تحميل فيديوهات TikTok الخاصة

الوصف:
عند محاولة تحميل فيديو من حساب TikTok خاص، يظهر خطأ.

الخطوات:
1. أرسل رابط فيديو من حساب خاص
2. اختر الجودة
3. لاحظ الخطأ

السلوك المتوقع:
يجب أن يتم تحميل الفيديو بنجاح أو عرض رسالة خطأ واضحة

المعلومات:
- Python: 3.9
- OS: Ubuntu 20.04
- الإصدار: 3.0.0
```

## 💡 اقتراح ميزات جديدة

### كيفية الاقتراح

1. **فتح Discussion:**
   - اذهب إلى "Discussions"
   - اضغط "New Discussion"
   - اختر "Ideas"

2. **اكتب الاقتراح:**
   - **العنوان**: اسم الميزة
   - **الوصف**: شرح الميزة
   - **الفائدة**: ماذا ستضيف
   - **الأمثلة**: حالات استخدام

### مثال اقتراح جيد:

```
العنوان: إضافة دعم تحميل البث المباشر (Live Streaming)

الوصف:
إضافة ميزة تحميل البث المباشر من YouTube و Twitch

الفائدة:
سيسمح للمستخدمين بتحميل البث المباشر بعد انتهاؤه

الأمثلة:
- تحميل بث YouTube المرتفع
- تحميل Twitch VODs
```
