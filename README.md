## 🔧 استكشاف الأخطاء

### خطأ: "BOT_TOKEN غير موجود"
**الحل:**
```bash
export BOT_TOKEN="your_token_here"
# أو أضفه في ملف .env
```

### خطأ: "منصة غير مدعومة"
**الأسباب:**
- الرابط غير صحيح
- المنصة غير مدعومة

**الحل:**
- تحقق من الرابط
- استخدم منصة مدعومة

### خطأ: "خطأ في تحميل فيديو TikTok"
**الأسباب:**
- الحساب خاص
- الفيديو محمي
- مشاكل في الاتصال

**الحل:**
- جرب مع فيديو من حساب عام
- تحقق من الاتصال بالإنترنت
- استخدم رابط مختلف

### خطأ: "حجم الملف يتجاوز الحد الأقصى"
**الحل:**
- اختر جودة أقل
- حمّل فيديو أقصر
- استخدم "صوت فقط" للموسيقى

### البوت لا يستجيب
**الحل:**
```bash
# تحقق من أن البوت يعمل
python bot.py

# تأكد من الإنترنت
ping 8.8.8.8

# أعد التشغيل
Ctrl+C
python bot.py
```

## 🌐 النشر والاستضافة

### على Railway (مجاني)
```bash
# تثبيت Railway CLI
npm install -g @railway/cli

# تسجيل الدخول
railway login

# نشر المشروع
railway up
```

### على Heroku
```bash
# تثبيت Heroku CLI
# ثم سجل دخول وأنشئ تطبيق
heroku login
heroku create your-bot-name

# عيّن المتغيرات
heroku config:set BOT_TOKEN="your_token"

# انشر المشروع
git push heroku main
```

### على Render
- انسخ مستودعك إلى GitHub
- ربط Render بحسابك
- أنشئ Web Service جديد
- عيّن BOT_TOKEN في البيئة
